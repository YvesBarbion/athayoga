Installing the sensor onto the base frame

  • Install the base frame onto the door.

    See Installing the base frame.

  • Drilling machine
  • Drill: size 10 mm
  • Pozidrive screwdriver for PZ1 screws
  • Sandpaper
  1. Drill through the 2 bases and the door using a 10-mm bit.
  2. Soften the edges using a sheet of sandpaper.
  3. Take the master-slave cable and pass it through the hole.
  4. Position the cable in the notch of the base and make sure it is firmly fixed.
  5. Remove the sensor cover.
    1. Put your finger in the hole as indicated in the picture.
    2. Pull firmly towards you in one movement.
  6. Pass the cable through the hole on the back of the sensor.
  7. Fasten the sensor on the base by sliding it downwards.
  8. Connect the black plug to the black connector.
  9. Safely tuck in all wires within the notch to avoid crushing them with the cover.
  10. Close the sensor which will not be connected to the door controller using a plug.
    Sensor connected to other module = SLAVE
  11. Fasten the lock screw firmly.
